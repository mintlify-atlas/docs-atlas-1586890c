# Welcome to MoodJourney

MoodJourney is a modern, digital and user-friendly journaling application that promotes self-reflection through features such as sentiment analysis and mood trend visualization. Built with Tauri (Rust backend + React frontend), it provides a secure and accessible platform for creating and storing journal entries.

## What you'll build

With MoodJourney, you can:

- Create and store secure journal entries with optional password protection
- Analyze your emotions using DistilRoBERTa sentiment analysis model
- Visualize mood trends through an interactive calendar interface
- Get AI-generated suggestions and feedback on your journal entries
- Use voice dictation to quickly capture your thoughts
- Chat with an AI assistant that understands your journaling context

## Technology stack

MoodJourney is built with modern technologies:

- **Frontend**: React 18 with Vite, Material-UI components, and Recharts for visualizations
- **Backend**: Rust with Tauri 2.5, providing native performance and security
- **Database**: SQLite with rusqlite for local data storage
- **AI Models**: 
  - WhisperAI for speech-to-text dictation
  - DistilRoBERTa for emotion classification
  - Google Gemini 2.0 Flash for AI assistance
